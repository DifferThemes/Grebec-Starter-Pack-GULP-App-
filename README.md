### Run Server

Set **ProxyServer** to true if you need proxy server \
Set **webpackModules** if you want to use ES6 imports
```js
const ProxyServer = false;
const webpackModules = true;
```

```bash
npm i
gulp
```


### Run Watcher

```bash
gulp
```

### Compile .SCSS

```bash
gulp scss
```

### Compile .JS

```bash
gulp js
```

### Compress images

```bash
gulp img
```
